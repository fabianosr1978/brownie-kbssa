Sistema offline para controle de Brownie do KBBSA.

Como usar:
1. Abra o arquivo index.html em qualquer navegador moderno.
2. Cadastre os produtos com custo unitário e fornecedor.
3. Registre compras de insumos com insumo/fornecedor, quantidade, valor unitário e local da compra.
4. Registre vendas com data, cliente, produto, quantidade e valor total calculado.
5. Informe o estoque inicial do mês na aba Inventário.
6. O dashboard mostra os totais de vendas, compras e relatórios por produto e cliente.

O sistema salva os dados localmente no navegador usando localStorage.
